## Virtual Environment

### Python Install:

	py -3 -m venv .venv

cmd.exe:
 	
	C:\> <venv>\Scripts\activate.bat

PowerShell:
	
 	.venv\Scripts\Activate.ps1

Bash:

 	source .venv/Scripts/activate

### UV Package Manager

```
uv init
uv venv
source .venv/Scripts/activate
```
